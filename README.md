Excel Requests: HTTP for Humans
===============================

Excel Requests is the only HTTP Addin for Excel, safe for human consumption.



Feature Support
---------------

Excel supports JSON via HTTP.



Excel Functions
---------------

- REQUESTS.GET(url, ...)
- REQUESTS.PUT(url, ...)
- REQUESTS.POST(url, ...)
- REQUESTS.PROPERTIES(url, ...)
